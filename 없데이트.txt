2021. 04. 01 - 소켓 통신 기초 / 센서 테스트
              자바로 소켓 통신을 열어 안드로이드 스튜디오 애플리케이션에서 데이터 전송 구현. (DodamDodam / socketOfJava)
              라즈베리파이-아두이노-아날로그 탁도 센서로 테스트. 오류. 시리얼 모니터에 값이 0.00만 출력. 일단 예제 코드 올려둠. (AnalogTurbiditySensorPrint)

2021. 04. 02 - 아날로그 탁도 센서의 플라스틱 부분을 벗겨내고 안쪽 나사를 조여서 출력전압 조절 후 다시 테스트.
              아날로그 탁도 센서 정상적으로 작동하는 것 확인.

2021. 04. 04 - 소켓 통신을 열어 클라이언트에서 서버로 데이터를 전송하고 서버가 해당 데이터를 다시 클라이언트에게 되돌려주는 소켓 통신 구현. (같은 PC에서 테스트)

2021. 04. 05 - 아날로그 탁도 센서로 커피와 물의 센서 값 차이 비교.
              라즈베리파이에서 소켓 서버를 열고 핸드폰의 안드로이드 앱이 클라이언트로써 데이터를 주고 받는 소켓 통신 테스트. (다른 PC, 같은 네트워크 대역)
              
2021. 04. 12 - 시리얼 통신을 이용해 아두이노 센서로부터 데이터 값을 받아와 라즈베리파이 프롬프트 창에 출력하기 (TemSocket)

2021. 04. 29 - 시리얼 통신을 이용해 아두이노 센서로부터 데이터 값을 받아와 라즈베리파이에서 가공하여 소켓 통신으로 안드로이드 애플리케이션까지 전송하기 완료!! (Socket_Temp)
